# Getting started with monorepo

## Description

Monorepo [TypeScript](https://www.typescriptlang.org/). The package manager is [pnpm](https://pnpm.io/). You need [NodeJs](https://nodejs.org/en/) >=18.12.0. This is an example of clean arquitecture in the frontend with React, Vue and Svelte.

## pnpm installation 
```bash
# enable corepack
$ corepack enable

# install pnpm
$ npm install -g pnpm
```

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Lint
```bash
# run lint with eslint
$ pnpm lint
```

## Format
```bash
# format all with prettier
$ pnpm format
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
